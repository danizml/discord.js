# Instalation

Node.js 16.9.0 or newer is required to run the bot;

```sh-session
npm install discord.js
yarn add discord.js
pnpm add discord.js
```
Also you need to create a config.json file and put;
```json
{
    "token": "Token here",
    "clientId": "Client ID here",
    "guildId": "Guild ID here"
}
```
